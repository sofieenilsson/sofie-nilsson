The Movember
Moustache Memory Game

Informative memory game to raise awareness about prostate cancer

Project

The Movember Moustache Memory Game is a cheerful and informative memory game. The game raises awerness about prostate cancer in a fun but yet informative way. The intended outcome is that the profit the game generates will be used to continuing the important research Movember supports about mens health.

Role: Art Director/Visual Designer

Responsibilities: Concept Development, Branding, Illustration, Packaging Design

Design

The game includes 30 tiles á 15 pairs, an informative leaflet/game board and packaging. The tile design consists of 15 different looking moustaches from around the world. This lets people relate to the game in a fun and cultural way. The leaflet communicates information about mens health in general and prostate cancer in particular. The other side of the leaflet consists of an overview of the moustaches that the players are supposed to pair up during the game, but also 'how to play' instructions.
