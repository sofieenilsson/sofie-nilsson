The Movember Moustache Memory Game

Informative memory game to raise awareness about prostate cancer

The Movember Moustache Memory Game is a fun, exciting and informative memory game. The game communicates a very serious subject such as cancer in a fun but yet informative way. The intended outcome is that a certain percent of the revenue stream from the game will be donated to the organization Movember.

The game includes 30 tiles á 15 pairs, an informative leaflet/game board and packaging. The tile design consists of 15 different looking moustaches from around the world. This lets people relate to the game in a fun and cultural way. The leaflet contains information about mens healt in general and prostate cancer in particular. The other side of the leaflet consists of an overview of the moustaches that the player are supposed to pair up during the game, but also 'how to play' instructions.
