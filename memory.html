The Movember Moustache Memory Game

Informative memory game to raise awareness about prostate cancer

The Movember Moustache Memory Game is a fun, exciting and informative memory game. The game communicates a very serious subject such as cancer in a fun but yet informative way. The intended outcome is that a certain percent of the revenue stream from the game will be donated to the organization Movember.

The game contains 30 tiles á 15 pairs, an informative leaflet/game board and packaging. In order to make it more fun the tile design contains of 15 different looking moustaches from around the world.

Fonts used:

Bebas Neue and Helvetica Neue
